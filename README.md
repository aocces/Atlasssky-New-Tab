\## 浏览器新标签页扩展



这是一个极简风格的浏览器新标签页扩展



\## 功能特点

\- 🎨 简洁美观的界面

\- 🔍 集成百度/谷歌/必应/duckduckgo搜索（按回车或点击按钮）

\- ⚡ 轻量快速，无多余权限



\## 如何安装

1\. 下载本仓库代码（Code -> Download ZIP）并解压。

2\. 打开 Chrome 浏览器，进入扩展管理页面（chrome://extensions/）。

3\. 开启“开发人员模式”。

4\. 点击“加载解压缩的扩展”，选择解压后的文件夹。

5\. 新标签页就会变成你的主页啦！
