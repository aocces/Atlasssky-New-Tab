\- ⚡ 轻量快速，无多余权限



如何安装

1\. 点击Release下载.crx或者压缩包格式的文件并下载。

2\. 打开 Chrome 浏览器，进入扩展管理页面（chrome://extensions/）。

3\. 开启“开发人员模式”。

4\. 压缩包文件先解压然后点击“加载解压缩的扩展”，选择解压后的文件夹；.crx格式的文件，请在在扩展管理界面将其拖入进去。

5\. 打开新标签页验证是否成功。



TIPS

.crx文件，插件暂时未上架Google插件商店，无法开启，请等待后续上架。
